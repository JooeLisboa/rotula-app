- [ ] Alternativas melhores
- [ ] Histórico e favoritos
- [ ] Paywall premium básico

- [ ] Descrição curta/longa

- [ ] Classificação indicativa
- [ ] Evidências de privacidade
- [ ] Assets localizados (pt-BR)
