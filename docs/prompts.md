# Prompts Prontos

## 1) UI Design
"Crie um design system mobile premium para o app Rótula (Brasil), com foco em scanner de produtos e score de qualidade. Entregue tokens, componentes, fluxo completo e protótipo navegável no Figma."

## 2) Desenvolvimento mobile
"Implemente o app Rótula com Expo Router + TypeScript, arquitetura por features, estados de loading/erro/vazio e integração com API REST versionada. Priorize scanner, busca e tela de resultado."

## 3) Backend/API
"Construa API NestJS para Rótula com módulos de auth, products, scans, favorites e subscriptions, PostgreSQL+Prisma, OpenAPI, RBAC admin e observabilidade com logs estruturados."

## 4) Painel admin
"Desenvolva painel admin web para Rótula com login seguro, dashboard de métricas, moderação de produtos enviados, edição de ingredientes e ajustes de scoring por categoria."

## 5) Landing page / waitlist
"Crie landing page de alta conversão para Rótula com proposta de valor clara, prova social, screenshots, formulário de waitlist e gatilho de convite com indicação."

## 6) Branding
"Defina identidade visual da marca Rótula com posicionamento de confiança e clareza, tom de voz humano, paleta premium e guideline para App Store/Play Store."
